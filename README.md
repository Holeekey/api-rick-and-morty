## 1. Installation

```bash
$ pnpm install
```

## 2. Configure Enviroment Variable

Use ".env.template" as a reference and name it ".env"

## 3. Start Docker

```bash
docker compose up -d
```

## 4. Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
